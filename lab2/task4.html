Дослідження методу console.log()

Відкрийте консоль браузера (F12 → Console) для перегляду результатів.

1. Вивід остачі від ділення

let a = 10;
let b = 3;
console.log(a % b);
console.log("Остача: " + (a % b));
console.log("Остача:", a % b);

2. Перевірка ділення без остачі

let x = 15;
let y = 5;
console.log(x % y === 0);
console.log("Ділиться без остачі: " + (x % y === 0));
console.log("Ділиться без остачі:", x % y === 0);

3. Вивід масиву

let arr = [10, 20, 30];
console.log(arr);
console.log("Масив: %s", arr);
console.log("Елементи: %d, %d, %d", arr[0], arr[1], arr[2]);

4. Комбінований приклад

let numbers = [7, 14, 21];
let n = 7;
for (let i = 0; i < numbers.length; i++) {
  console.log(numbers[i] + " % " + n + " = " + (numbers[i] % n));
}
